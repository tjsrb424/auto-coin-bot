## Visual Notes

- 어두운 트레이딩 터미널 톤을 기본으로 한다.
- 좌측 고정 사이드바, 상단 상태/검색 바, 카드형 섹션 구성을 공통 레이아웃으로 사용한다.
- 강조색은 보라색을 기본 액션/선택 상태에 사용하고, 수익/정상은 초록색, 손실/위험은 빨간색, 경고는 노란색 계열로 구분한다.
- 숫자, 가격, 수량, 수익률은 우측 정렬과 tabular/monospace 스타일을 우선한다.
- 실거래 관련 UI는 항상 상태, 위험, 주문 결과를 명확히 보여준다.
